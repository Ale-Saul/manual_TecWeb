Manual html

Nombre del Elemento Descripcion Uso
<html> El elemento raíz que encierra todo el documento HTML Define el comienzo de un documento HTML
<head> Contiene información sobre el documento HTML, como metadatos, scripts y enlaces a hojas de estilo Define una sección de encabezado para el documento HTML
<body> Contiene el contenido del documento HTML, como texto, imágenes, enlaces, formularios, etc. Define el cuerpo principal del documento HTML
<div> Se utiliza para agrupar elementos HTML juntos y aplicar estilos a ellos con CSS Define una sección genérica o contenedor sin ningún significado especial
<p> Se utiliza para crear un párrafo de texto en el documento HTML Define un párrafo de texto
<img> Se utiliza para incrustar una imagen en el documento HTML Define una imagen en el documento HTML
<a> Se utiliza para crear un enlace a otra página web, archivo o ubicación en la misma página Define un hipervínculo
<form> Se utiliza para crear un formulario en el documento HTML que permite a los usuarios enviar información al servidor web Define un formulario HTML para entrada de datos del usuario
<input> Se utiliza para crear campos de entrada en un formulario, como cajas de texto, botones de opción y botones de envío Define un elemento de entrada para un formulario HTML
<meta> Se utiliza para incluir metadatos en el documento HTML, como el autor, la descripción y palabras clave Define metadatos para el documento HTML
<title> Se utiliza para especificar el título del documento HTML, que se muestra en la pestaña del navegador Define el título del documento HTML
<h1> a <h6> Se utilizan para crear encabezados de sección en el documento HTML, donde <h1> es el encabezado más importante y <h6> es el menos importante Define un encabezado de sección
<span> Se utiliza para agrupar elementos HTML juntos y aplicar estilos a ellos con CSS Define una sección genérica o contenedor sin ningún significado especial
<article> Se utiliza para marcar el contenido independiente y autónomo en el documento HTML, como un artículo de periódico o un blog Define un artículo independiente en el documento HTML
<section> Se utiliza para agrupar contenido relacionado en el documento HTML, como secciones de un artículo o capítulos de un libro Define una sección lógica en el documento HTML
<nav> Se utiliza para definir un conjunto de enlaces de navegación en el documento HTML Define una sección de navegación en el documento HTML
<aside> Se utiliza para marcar el contenido relacionado pero separado del contenido principal del documento HTML, como una barra lateral o un cuadro de anuncios Define una sección de contenido relacionado pero separado del contenido principal del documento HTML
<footer> Se utiliza para definir el pie de página del documento HTML Define el pie de página del documento HTML
