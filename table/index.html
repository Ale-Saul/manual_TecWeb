Nombre del Elemento Descripcion Uso
<html> Define el comienzo de un documento HTML El elemento raíz que encierra todo el documento HTML
<head> Define una sección de encabezado para el documento HTML Contiene información sobre el documento HTML, como metadatos, scripts y enlaces a hojas de estilo
<body> Define el cuerpo principal del documento HTML Contiene el contenido del documento HTML, como texto, imágenes, enlaces, formularios, etc.
<div> Define una sección genérica o contenedor sin ningún significado especial Se utiliza para agrupar elementos HTML juntos y aplicar estilos a ellos con CSS
<p> Define un párrafo de texto Se utiliza para crear un párrafo de texto en el documento HTML
<img> Define una imagen en el documento HTML Se utiliza para incrustar una imagen en el documento HTML
<a> Define un hipervínculo Se utiliza para crear un enlace a otra página web, archivo o ubicación en la misma página
<form> Define un formulario HTML para entrada de datos del usuario Se utiliza para crear un formulario en el documento HTML que permite a los usuarios enviar información al servidor web
<input> Define un elemento de entrada para un formulario HTML Se utiliza para crear campos de entrada en un formulario, como cajas de texto, botones de opción y botones de envío
<meta> Define metadatos para el documento HTML Se utiliza para incluir metadatos en el documento HTML, como el autor, la descripción y palabras clave
<title> Define el título del documento HTML Se utiliza para especificar el título del documento HTML, que se muestra en la pestaña del navegador
<h1> a <h6> Define un encabezado de sección Se utilizan para crear encabezados de sección en el documento HTML, donde <h1> es el encabezado más importante y <h6> es el menos importante
<span> Define una sección genérica o contenedor sin ningún significado especial Se utiliza para agrupar elementos HTML juntos y aplicar estilos a ellos con CSS
<article> Define un artículo independiente en el documento HTML Se utiliza para marcar el contenido independiente y autónomo en el documento HTML, como un artículo de periódico o un blog
<section> Define una sección lógica en el documento HTML Se utiliza para agrupar contenido relacionado en el documento HTML, como secciones de un artículo o capítulos de un libro
<nav> Define una sección de navegación en el documento HTML Se utiliza para definir un conjunto de enlaces de navegación en el documento HTML
<aside> Define una sección de contenido relacionado pero separado del contenido principal del documento HTML Se utiliza para marcar el contenido relacionado pero separado del contenido principal del documento HTML, como una barra lateral o un cuadro de anuncios
<footer> Define el pie de página del documento HTML Se utiliza para definir el pie de página del documento HTML
<header> Define la cabecera de una página o sección. Se utiliza para agregar elementos de encabezado en una página web, como el título, logo, menú de navegación, entre otros.
<style> Define una sección de estilo en el documento HTML. Se utiliza para agregar estilos CSS en una página web, como el color de fondo, color de texto, tamaño de fuente, entre otros.
<nav> Define un menú de navegación. Se utiliza para agregar un menú de navegación en una página web, que permite a los usuarios navegar por diferentes secciones o páginas del sitio.
<main> Define el contenido principal de una página. Se utiliza para agregar el contenido principal de una página web, como artículos, secciones, formularios, entre otros.
<section> Define una sección en una página o artículo. Se utiliza para dividir el contenido de una página o artículo en secciones temáticas o de contenido.
<article> Define un artículo independiente en una página. Se utiliza para marcar un contenido independiente y completo dentro de una página, como un post en un blog o una noticia en un sitio de noticias.
<aside> Define un contenido relacionado con la página pero no es parte del contenido principal. Se utiliza para agregar contenido relacionado, como una barra lateral o una sección de publicidad, que no es parte del contenido principal de la página.
<br> Inserta un salto de línea. Se utiliza para agregar un salto de línea en un texto o elemento en HTML.
<ol> Define una lista ordenada. Se utiliza para agregar una lista de elementos en un orden específico, como pasos a seguir o una lista numerada.
<li> Define un elemento de una lista. Se utiliza dentro de una lista ordenada o no ordenada para definir un elemento de la lista.
<ul> Define una lista no ordenada. Se utiliza para agregar una lista de elementos en cualquier orden, como una lista de elementos sin importancia o una lista de opciones.
<hr> Inserta una línea Se utiliza para agregar una línea horizontal en un documento HTML, que puede ser utilizada para separar secciones o elementos.
<wbr> Define un posible lugar de ruptura de línea en un texto. Se utiliza para indicar un posible lugar donde un texto se puede dividir en varias líneas si es necesario, como en palabras largas o en URLs.
<sup> Define un texto sobrescrito. Se utiliza para indicar un texto que debe aparecer en una posición superior a la línea de texto normal, como en una fórmula matemática o un exponente.
<sub> Define un texto subescrito. Se utiliza para indicar un texto que debe aparecer en una posición inferior a la línea de texto normal, como en una fórmula química o una nota a pie de página.
<strong> Define un texto importante. Se utiliza para marcar un texto que es importante o relevante dentro de un contenido, como en un titular o una palabra clave.
<em> Define un texto enfatizado. Se utiliza para marcar un texto que debe ser enfatizado o destacado, como en una cita o una palabra clave.
<mark> Define un texto resaltado. Se utiliza para marcar un texto que debe ser resaltado o destacado de alguna manera, como en un término de búsqueda o un error en un formulario.
<q> Define una cita corta. Se utiliza para marcar una cita breve dentro de un texto, como una frase o un enunciado que se ha tomado de otra fuente.
<blockquote> Define una cita larga. Se utiliza para marcar una cita que ocupa varios párrafos o líneas de texto, como un pasaje de un libro o un artículo de revista.
<cite> Define una referencia a una fuente. Se utiliza para marcar una referencia a una fuente, como un libro, un artículo o un sitio web, que se ha utilizado en el contenido.
<dfn> Define un término. Se utiliza para marcar un término que se está definiendo en el contenido, como un concepto o una palabra técnica.
<abbr> Define una abreviatura. Se utiliza para marcar una abreviatura en el contenido, como "Mr." para "Mister" o "NASA" para "National Aeronautics and Space Administration".
<small> Define un texto pequeño. Se utiliza para indicar un texto que debe aparecer en una fuente más pequeña que el texto circundante, como en una nota al pie o un aviso legal.
<ins> Define un texto insertado. Se utiliza para indicar un texto que se ha añadido recientemente al contenido, como en un chat o un historial de cambios.
<del> Define un texto eliminado. Se utiliza para indicar un texto que se ha eliminado o modificado en el contenido, como en un historial de cambios o una corrección de errores.
<s> Define un texto tachado. Se utiliza para indicar un texto que se ha tachado o marcado como obsoleto o incorrecto en el contenido, como en un historial de cambios o una corrección de errores.
<code> Define código de computadora. Se utiliza para marcar un fragmento de código dentro del texto.
<samp> Define un ejemplo de salida. Se utiliza para marcar un ejemplo de salida en el contenido, como la salida de un programa o un comando de línea de comandos.
<kbd> Define entrada del teclado. Se utiliza para marcar la entrada del teclado en el contenido, como la pulsación de una tecla o la combinación de varias teclas.
<pre> Define texto preformateado. Se utiliza para marcar un bloque de texto que debe aparecer en su formato original, sin procesar ni modificar, como el código de programación o una transcripción de un discurso.
<address> Define información de contacto. Se utiliza para marcar información de contacto en el contenido, como el nombre, la dirección y el número de teléfono de una persona o una empresa.
&lt; Menor que. Se utiliza para mostrar el símbolo < en el contenido, sin que sea interpretado como un elemento HTML.
&gt; Mayor que. Se utiliza para mostrar el símbolo > en el contenido, sin que sea interpretado como un elemento HTML.
&amp; Ampersand. Se utiliza para mostrar el símbolo & en el contenido, sin que sea interpretado como un carácter especial.
&quot; Comillas dobles. Se utiliza para mostrar comillas dobles en el contenido, sin que sean interpretadas como delimitadores de texto.
&apos; Comillas simples. Se utiliza para mostrar comillas simples en el contenido, sin que sean interpretadas como delimitadores de texto.
<table> Define una tabla. Se utiliza para crear una tabla en HTML, que puede contener filas y columnas de datos.
<thead> Define la sección de encabezado de una tabla. Se utiliza para indicar la sección de encabezado de una tabla, que suele contener las etiquetas <th> para cada columna de la tabla.
<tbody> Define la sección de cuerpo de una tabla. Se utiliza para indicar la sección de cuerpo de una tabla, que suele contener las etiquetas <tr> para cada fila de la tabla.
<tfoot> Define la sección de pie de una tabla. Se utiliza para indicar la sección de pie de una tabla, que suele contener las etiquetas <th> o <td> para cada columna de la tabla, y se coloca al final del elemento <table>.
<tr> Define una fila de una tabla. Se utiliza para crear una fila en una tabla, que suele contener las etiquetas <th> o <td> para cada celda de la fila.
<th> Define una celda de encabezado en una tabla. Se utiliza para crear una celda de encabezado en una tabla, que se coloca dentro de la etiqueta <thead> o <tfoot>, y suele contener el texto de la columna.
<td> Define una celda de datos en una tabla. Se utiliza para crear una celda de datos en una tabla, que se coloca dentro de la etiqueta <tbody>, y suele contener el valor de la celda.
